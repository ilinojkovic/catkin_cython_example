# Catkin Cython Example

### Run
```
$ mkdir -p workspace/src
$ cd workspace/src
$ git clone git@github.com:ilinojkovic/catkin_cython_example.git
$ cd ..
$ catkin_make
$ rosrun catkin_cython_example demo_example_a.py
```
